# CodeApp Template Repos

A collection of template repositories for building AI-powered developer tools and cloud-native applications.

## Templates

### [context-window](./context-window)
A toolkit for scanning, parsing, and compressing codebases into LLM-friendly context. Helps you fit more relevant code into your AI assistant's context window.

### [kube-dashboard-lite](./kube-dashboard-lite)
A lightweight Kubernetes dashboard built with Go. Provides a simple web UI for viewing and managing cluster resources without the complexity of the full Kubernetes Dashboard.

### [llm-gateway](./llm-gateway)
A unified API gateway for multiple LLM providers (OpenAI, Anthropic, etc.). Features request routing, caching, rate limiting, and metrics collection.

### [prompt-forge](./prompt-forge)
A prompt management and testing framework. Define prompts in YAML, use template variables, and run automated tests to ensure prompt quality.

## Getting Started

Each template is a standalone project. Navigate to the template directory and follow its README for setup instructions.
